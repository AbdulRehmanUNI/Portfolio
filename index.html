Conway's Game of Life

   

I created a web application for Conway's Game of Life using ReactJS. The Game of Life is a cellular automaton devised by the British mathematician John Horton Conway in 1970. The game is a zero-player game, meaning that its evolution is determined by its initial state. The game is hosted on GitHub Pages.
